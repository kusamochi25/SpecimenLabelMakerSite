使用方法

Modus operandi

掲載日 2026年6月20日

ラベルを作成する操作は共通ですが、PDF保存と下書き件数は購入状態によって経路が異なります。最初に自分の状態を確認し、該当する分岐を参照してください。

Guide 01

最初に確認する4つの経路

  • 購入なし: PDFは月2回まで無料保存できます。保存時に通信と広告視聴が必要です。下書きは1件です。
  • 月額または年額プレミアム: PDFを回数制限なし、広告なし、オフラインで保存できます。下書きは、下書き保存枠追加を別途購入していなければ1件です。
  • 下書き保存枠追加のみ購入: 下書きは10件になります。PDF保存は無料利用と同じ経路で、月2回、通信と広告視聴が必要です。
  • プレミアムと下書き保存枠追加の両方を購入: PDFはプレミアム経路で保存し、下書きは10件まで利用できます。

月額プレミアムと年額プレミアムの機能は同じです。異なるのは請求期間です。下書き保存枠追加は、プレミアムとは別の買い切り商品です。

Guide 02

共通のラベル作成手順

  1. ホーム画面で用紙が「A4」であることを確認し、「新規作成」を選びます。
  2. プレビュー画面右下の「＋」を押します。
  3. データラベル、同定ラベル、コレクションラベル、無地ラベルから種類を選びます。
  4. 必要事項、ラベル寸法、文字サイズ、作成枚数を入力し、「プレビューに追加」を押します。
  5. 必要なラベルを同じ操作で追加し、A4プレビューで配置を確認します。

1つのPDFファイルにはA4縦の1ページだけを作成します。指定枚数が1ページに収まらない場合は、収まる枚数だけが追加され、アプリが調整結果を表示します。

Guide 03

追加後の編集と設定

プレビュー上のラベルをタップすると、次の操作を選べます。

  • 同じ内容のラベルをすべて編集する
  • 選択した1セルだけを編集する
  • 同じ内容を指定枚数だけコピー追加する
  • 同じ内容のラベルをすべて削除する
  • 選択した1セルだけを削除する

プレビュー上部の「設定」では、すべてのセルへ適用する外枠の太さを変更できます。ラベルの追加、編集、削除、コピー、設定変更は下書きへ自動保存されます。

データラベルの「改行設定」では、採集日、採集者、緯度・経度、標高、採集方法、備考について、各項目の前で改行するか個別に切り替えられます。国と産地・採集地は必ず改行されます。改行設定はラベルごとに保存され、下書きとPDF履歴から復元した場合も引き継がれます。

採集日は外国式と日本式から選択できます。外国式は14.JUN.2026、日本式は2026.JUN.14のように、同じ月表記と区切りで順序だけを反転します。期間、月まで、旬にも同じ順序が適用され、旬は日本式では初旬、中旬、下旬と表記します。英語の月表記はJANからDECまでの3文字です。座標表記はlat./lon.、方位記号（°N/S/E/W）、日本語（北緯・南緯・東経・西経）から選択できます。方位は入力値の符号から変換され、緯度は＋が北、−が南、経度は＋が東、−が西です。

Guide 04

分岐A: 無料でPDFを保存する

  1. プレビュー右上の「PDF保存」を押します。
  2. 文字切れや学名表記などの警告が表示された場合は、内容を確認して修正するか、「このまま保存」を選びます。
  3. アプリが通信し、当月の無料保存枠と利用可能状態を確認します。
  4. 確認画面で残り回数を確認し、「広告を表示して保存へ進む」を押します。
  5. 広告を最後まで視聴します。広告報酬が確認されると無料保存枠を1回使用し、PDF保存へ進みます。
  6. 「PDFを保存しました」と表示されたら完了です。

無料保存にはインターネット接続、無料保存APIの確認、広告の読み込みと視聴完了が必要です。広告を途中で閉じた場合、広告を読み込めない場合、無料枠を使い切った場合は保存できません。初回などには、広告に関する同意画面が先に表示される場合があります。

無料枠の予約後にPDF生成または履歴保存が失敗した場合、アプリは無料枠の復元を試みます。復元結果は画面の案内で確認してください。

Guide 05

分岐B: プレミアムでPDFを保存する

  1. 月額または年額プレミアムが「加入中」であることをホーム画面または設定画面で確認します。
  2. プレビュー右上の「PDF保存」を押します。
  3. 保存前の警告があれば確認します。
  4. 広告表示と無料保存枠の確認を行わず、PDFが端末へ直接保存されます。

プレミアムではPDF保存回数の制限がなく、オフラインでも保存できます。ただし、購入状態の取得、購入、復元などStoreKitを利用する操作には通信が必要になる場合があります。

Guide 06

下書きの分岐と上書き

  • 下書き保存枠追加が未購入: 下書きは1件です。
  • 下書き保存枠追加を購入済み: 下書きは10件です。プレミアム加入の有無は、この件数へ影響しません。

最初のラベルを追加すると下書きが作成され、その後の変更は自動保存されます。上限に達した状態で「新規作成」を選ぶと警告が表示され、続行した場合は最も古い下書きが削除されて新しい下書きへ置き換わります。

「下書きから再開」では保存中の編集内容を開けます。下書きを削除しても、すでに保存したPDFファイルは削除されません。

Guide 07

購入と復元

  1. ホーム画面右上の歯車を開きます。
  2. 「購入と復元」を選びます。
  3. 月額プレミアム、年額プレミアム、下書き保存枠追加から必要な商品を選び、App Storeの購入画面で確定します。

機種変更や再インストール後は、同じApple Accountで「購入を復元」を選択してください。プレミアムの有効期間が終了するとPDF保存は無料経路へ戻ります。買い切りの下書き保存枠追加は、購入情報が有効である限り10件のままです。

Guide 08

保存後の確認、共有、再編集

  • 保存済みPDF: PDFファイルを開き、共有または削除できます。
  • PDF履歴: 保存時のラベル構成をプレビューへ復元し、編集して新しいPDFを作成できます。
  • Filesアプリ: 「標本ラベルメーカー」フォルダからPDFファイルを確認できます。

PDF履歴から削除するとPDFファイル本体も削除されます。保存済みPDF画面で削除した場合も、対応する履歴が整理されます。必要なPDFは削除前に共有または別の場所へ保存してください。

Guide 09

印刷前の確認

印刷時はA4用紙を選び、倍率を「実際のサイズ」または「100%」に設定してください。「用紙に合わせる」などの自動拡大縮小が有効だと、指定したラベル寸法と異なる場合があります。

学名、採集地、採集者、同定情報、文字切れを確認し、標本へ使用する前に必ず試し印刷を行ってください。PDFの解像度と内部変換については、PDFと解像度を参照してください。
